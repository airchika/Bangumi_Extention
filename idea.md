## 两人共同维度（新增候选）

- [ ] 共同满分 — 双方都给了 10 分的条目，按收藏人数降序（看你们共同的满分神作有多少人也认可）
- [ ] 共同零分 — 双方都给了 1 分的条目，按收藏人数降序（看你们共同的粪作有多少人也踩）
- [ ] 评分完全一致 — 两人给了完全相同分数的条目，按分数降序，同分按收藏人数
- [ ] 分歧反转 — 两人差异大且方向相反：一人高于大众、一人低于大众（公式：`(我-大众) * (对方-大众)` 越负越优先）
- [ ] 神作共鸣 — 双方都给了 ≥8 分且大众均分也 ≥8 的条目（真正的"大家都觉得好"的共识）
- [ ] 雷作共鸣 — 双方都给了 ≤3 分且大众均分也 ≤5 的条目（真正的"大家都觉得差"的共识）
- [ ] 冷门遗珠 — 双方都给了 ≥8 分但收藏人数 < 500 的条目（你们共同发现的小众宝藏）
- [X] 热门分歧 — 收藏人数 > 5000 但双方评分较大的条目（大热门但你俩看法相反）
- [X] 共同回忆 — 双方都看过的老番（按条目日期升序，从最早的作品开始）
- [X] 共同追新 — 双方都看过的近作（按条目日期降序，从最新的作品开始）
